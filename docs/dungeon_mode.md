以下の追加機能のための設計書を作成してください

・中1,2,3各ワールドの最後にそのワールドのすべての問題からランダムに出題されるまとめステージを作成する

・別モードである「ダンジョンモード」を実装する。

・ダンジョンモードはクイズバトルの出題範囲を全ワールドの全単元とし、敵も強化した進化系であり、初めはHP10,攻撃力1の状態でスタートし、HPが0になって敗北するまで戦闘を続けるエンドレスモード。

・戦闘中、敵はカウントを持ち、1秒ごとにカウントは減少、0になると自分は攻撃を受け敵のカウントは再び最初に戻る。

・クイズバトル同様、回答のミスをすると相手の攻撃と同じダメージを受ける、この際相手のカウントに変化はない。

・クイズバトル同様コンボシステムを採用、コンボが続く限り {100 + (コンボ数*10)}%の攻撃補正がかかり、最大で攻撃力は200%になる。

・戦闘終了時にランダムな数のG（ゴールド）を獲得する、ゴールドの数は階を重ねるほどもらえる（後述のアイテムの金額を踏まえて適切なドロップ量の提示を期待する）

・戦闘が終了すると次の階へ進める。この際次の階に行ってもアイテムの効果以外ではHPは回復しない。

・階数が増えるほど敵のHPと攻撃力は増加する

・3回戦闘を終了するたびにSHOPが現れ、Gでアイテムを購入できる

・SHOPで購入できるアイテムは「こうげきアップグレード」（攻撃力を2増加、初期値段10Gだが購入するたびに5G値上がりする）、「ぼうぎょアップグレード」（HPを5増加、初期値段10Gだが購入するたびに5G値上がりする）、「かいふくセット」（HPを全回復する、初期値段30Gだが購入するたびに20G値上がりする）、「そうび」（保持している間戦闘で効果を発揮するアイテム、3個まで保有可能で不要になったそうびは「すてる」ことができる、毎回のSHOPでランダムに選ばれた3つを購入できる。抽選確率はノーマル50%、レア30%、スーパーレア15%、レジェンド5％で出現する。現在所持しているアイテムはSHOPに並ばない）
